# Spring Boot + PostgreSQL + Tailwind CSS + Alpine.js Contact Form

A full-stack web application with:
- **Spring Boot** (Java backend)
- **PostgreSQL** (database)
- **Tailwind CSS** + **Alpine.js** (frontend UI)

## 🧱 Features

- Responsive contact form UI (Tailwind + Alpine.js)
- Backend API to save form submissions
- Data persisted in PostgreSQL
- Validation on frontend and backend
- JSON-based API
